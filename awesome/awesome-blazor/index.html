    Introduction¶

    What is Blazor?¶

    Blazor is a .NET web framework to build client web apps with C#.

    Blazor lets you build interactive web UIs using C# instead of JavaScript. Blazor apps are composed of reusable web UI components implemented using C#, HTML, and CSS. Both client and server code is written in C#, allowing you to share code and libraries. 有关的更多信息 official Blazor website.

    Get started¶

    To get started with Blazor, follow the instructions in the Blazor Get Started 文档.
